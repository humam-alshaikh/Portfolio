Comparative Study of Model Order Reduction Methods

Bachelor's Graduation Project · GJU / Robert Bosch GmbH
structural dynamics

Problem

Full FE models of jointed structures often contain over 100,000 DOFs, making repeated dynamic analyses computationally expensive. The goal was to determine which linear model order reduction method best preserves structural dynamics at a fixed reduction level.

Approach

Implemented Guyan static condensation, Craig–Bampton fixed-interface CMS, and free-interface (Herting) CMS in ANSYS Mechanical. To ensure a fair comparison, all three methods used the same interface (793 master nodes across four interfaces) and the same reduction level (20 generalized coordinates). We checked the mesh convergence, and then the reduced models were evaluated using modal, harmonic (FRF, 0–2,500 Hz), and transient analyses.

Tools

ANSYS Mechanical · ANSYS APDL (automation, post-processing) · Python (MAC from directional nodal deformations).

Finite Element Solution of a Gravity-Loaded Beam Using Q4 Elements

Numerical Mechanics I · University of Kassel (SS2025)
computational mechanics

Problem

Compute the deformation and stress distribution of a beam under self-weight using 2D finite element solver, then verify the results against Euler–Bernoulli beam theory.

Approach

Developed a 2D finite element solver from scratch using bilinear quadrilateral (Q4) elements. Implemented element formulation, Gauss integration, global stiffness assembly, gravity loading, and stress recovery. Generated structured meshes and evaluated both clamped–clamped and pinned–pinned beams with different slenderness ratios (L/H = 10 and 100) before validating the results against analytical beam theory.

Tools

Python · NumPy (mesh, assembly, linear solve) · pandas · matplotlib (stress visualization on the deformed mesh).

Tracing Gas Flow & Thermal Characteristics via Aerodynamic Analysis

Robert Bosch GmbH · Internship Report
thermo-fluids

Problem

Understand and predict liquid-water formation at the anode inlet of a passive hydrogen fuel cell under varying flow, pressure, and humidity conditions.

Approach

Developed a simplified 1D compressible-flow model based on the conservation laws. Computed Mach number, modeled contractions, expansions, and choked flow, then solved the downstream flow state from the continuity, momentum, and energy equations. Finally, predicted gas composition using Faraday's law and calculated the mixed-gas temperature.

Tools

Python · OpenCV · SciPy (least_squares / TRF) · CoolProp (IAPWS-95), with a computer-vision module (SIFT, FLANN, RANSAC homography) and a synchronization layer for multi-rate sensor data.

Results

  • Worked on a computer vision pipeline that detects observation windows in experimental videos within milliseconds per frame.
  • Predicts downstream flow (velocity, pressure, temperature, Mach number), including choked cases.
  • Estimates gas concentration and condensation from measured data across cell stack.
  • Replaced an earlier two-experiment "delta" method with a single-pass nonlinear solve.
